# 🎨 Air Draw WebApp

Virtual drawing application that enables users to draw, write, and erase using hand gestures in real time.

## 🚀 Live Demo

https://ai-draw-webapp.onrender.com

## 📂 GitHub Repository

https://github.com/varsha490/ai-draw-webapp

## ✨ Features

* Draw using hand gestures
* Write names, symbols, and shapes
* Erase drawings interactively
* Real-time virtual canvas
* User-friendly interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* MediaPipe
* OpenCV
